# Contributing Guide

See [CONTRIBUTING.md](../CONTRIBUTING.md) in the root of the repository for the full contributing guide.

## Quick Reference

| Task | Command |
|------|---------|
| Install deps | `npm install` |
| Run tests | `npm test` |
| Lint code | `npm run lint` |
| Format code | `npx prettier --write src/` |
| Build | `npm run build` |

## Branch Naming

| Type | Pattern | Example |
|------|---------|---------|
| Feature | `feature/<description>` | `feature/add-interstitial-analytics` |
| Bug fix | `fix/<description>` | `fix/banner-dismiss-on-mobile` |
| Docs | `docs/<description>` | `docs/update-api-reference` |
| Release | `release/<version>` | `release/2.0.0` |
