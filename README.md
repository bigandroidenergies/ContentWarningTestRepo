# ContentWarningTestRepo

> **Test repository for verifying interstitial and banner content warning UI in a mobile environment.**

This repository is intentionally populated with sample data to exercise common navigation paths and content surfaces, including issues, pull requests, actions, discussions, releases, and more.

---

## 🚀 Features

- Sample CI/CD workflows (GitHub Actions)
- Example source code in JavaScript/TypeScript
- Unit and end-to-end test suites
- Contributing guidelines and templates
- MIT License

## 📁 Project Structure

```
ContentWarningTestRepo/
├── .github/
│   ├── workflows/       # GitHub Actions CI/CD pipelines
│   └── ISSUE_TEMPLATE/  # Issue form templates
├── src/
│   ├── components/      # UI components
│   ├── utils/           # Utility functions
│   └── api/             # API client layer
├── tests/
│   ├── unit/            # Unit tests
│   └── e2e/             # End-to-end tests
└── docs/                # Additional documentation
```

## 🛠️ Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```
